## Objective
The objective of this task is to analyze customer data for a subscription-based business in order to:
- Identify customer churn patterns
- Analyze retention trends
- Understand key drivers affecting customer lifetime value
- Provide actionable recommendations to reduce customer churn
##  Dataset
- File: `Task2_Customer_Churn_Dataset.xlsx`
- Description: Subscription-based customer dataset containing customer tenure, plan type, spending, and churn status.
### Key Columns
- CustomerID  
- JoinDate  
- TenureMonths  
- PlanType  
- MonthlyFee  
- TotalSpend  
- LastActiveMonth  
- Churn (Yes / No)
##  Tools Used
- Power BI  
- Microsoft Excel  
##  Key KPIs
- Total Customers  
- Churn Rate (%)  
- Retention Rate (%)  
- Average Customer Tenure  
- Average Customer Lifetime Value (CLV)  
##  Dashboard Visualizations
- KPI Cards (Churn Rate, Retention Rate, CLV, etc.)
- Churn by Plan Type (Column Chart)
- Churn vs Tenure (Bar Chart)
- Retention Trend Over Time (Line Chart)
- Churn Distribution (Pie Chart)
- Interactive Slicer by Plan Type
